Dobrodošli u Game Asset Store, web aplikaciju razvijenu u .NET okruženju koja omogućava developerima kupovinu i prodaju digitalnih asseta za igre. Ovaj projekat demonstrira integraciju eksternih API-jeva za automatizaciju workflow-a i sigurnu obradu plaćanja.

Funkcionalnosti: 
GitHub API Integracija: Umjesto klasičnog file servera, kupljeni asseti se automatski uploaduju i čuvaju na privatnim GitHub repozitorijumima radi verzionisanja i sigurnosti.

Stripe Payment Gateway: Potpuno funkcionalan sistem plaćanja koji podržava kartice i generisanje računa.

Asset Management: Korisnici mogu pregledati, filtrirati i kupovati 2D/3D modele, skripte i zvučne efekte.

Baza Podataka: Hostovana na SmarterASP.NET (SQL Server), zadužena za korisničke podatke, katalog proizvoda i istoriju transakcija.

Tehnologije:
Backend: .NET (ASP.NET Core MVC / Web API)

Baza podataka: Microsoft SQL Server (SmarterASP.NET hosting)

Plaćanje: Stripe API

Storage & Versioning: GitHub API (Octokit)

Frontend: HTML5, CSS3, JavaScript (Bootstrap/Razor Pages)
